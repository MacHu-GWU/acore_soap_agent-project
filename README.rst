Install
------------------------------------------------------------------------------

``acore_soap_agent`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install acore-soap-agent

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade acore-soap-agent
